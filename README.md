# mememe [![Build Status](https://travis-ci.org/nervetattoo/mememe.svg?branch=master)](https://travis-ci.org/nervetattoo/mememe)

> Me


## Install

```
$ npm i --save mememe
```


## Usage

```js
var mememe = require('mememe');

var me = 'me'

mememe(me);
//=> 'me'

mememe('me', 'you');
//=> 'me'
```
